## ðŸ’¡ Project Overview

**OpenAPI Generator Gauge Plugin** is a custom plugin for the [OpenAPI Generator](https://openapi-generator.tech/) that integrates [Gauge](https://docs.gauge.org/writing-specifications) for API testing. This plugin provides a collection of templates that enable the generation of Gauge test specifications for C#, Python, Go, and Java. The generated tests can be executed against any client library generated by OpenAPI Generator, providing a flexible and language-agnostic testing solution for your APIs.

## ðŸŽ¯ Requirements
To use this plugin, you'll need the following:

1. **[OpenAPI Generator](https://openapi-generator.tech/docs/installation)** version **7.11.0** or later.
2. **[Gauge](https://docs.gauge.org/getting_started/installing-gauge)** installed on your machine.

Ensure that both OpenAPI Generator and Gauge are correctly set up before attempting to use this plugin.
